## 2. 근본 원인 분석 및 재발 방지

### 근본 원인
교재 생성 프롬프트에 **"이 교재는 [주제] 12주 단독 시리즈"** 임을 명시하지 않아, AI가 4차원 영성 전체 시리즈 순서(생각→꿈→말→믿음)로 "다음 주 예고"를 생성함.

### 재발 방지 조치

1. **프롬프트 개선** (즉시 적용)
   - 교재 생성 프롬프트에 명시: *"이 교재는 [주제]만을 12주 동안 다루는 독립 시리즈입니다. '다음 주 예고'는 반드시 이 교재의 실제 다음 주차 제목을 참조하십시오."*

2. **생성 후 자동 검증 스크립트** (다음 Do 이터레이션)
   - 교재 생성 완료 후 "다음 주 예고" vs 실제 다음 주차 제목 자동 비교 스크립트 추가

3. **Check 단계 필수 항목** (Plan 문서에 이미 반영)
   - 교재 생성 직후 Check 단계에서 일관성 항목 전수 검증
