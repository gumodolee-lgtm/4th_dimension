> **Status**: In Progress

## 1. 프로젝트 배경 및 목표

### 배경
조용기 목사님의 1,000편 이상 설교 txt 파일을 보유하고 있으며,
이를 체계적으로 분류·가공하여 **4차원의 영성** 원리에 기반한
전문 성경공부 교재를 제작하고자 한다.

### 핵심 목표
- **설교 자동 분류**: 1,000편+ 설교를 4차원 요소별로 AI 분류
- **전문가 수준 교재**: 신학적 깊이와 실용성을 겸비한 12주 커리큘럼
- **확장 가능한 시스템**: 4개 요소 각각의 교재 독립 생성 가능

## 3. 시스템 구성 (2단계)

### Phase A: 설교 분류 시스템
```
입력: sermons/*.txt (1,000편+)
처리: Claude API 자동 분류
출력: classified/{요소}/{설교명}.json
```

**분류 기준**:
- 주요 요소 (Primary): 설교의 핵심 메시지
- 보조 요소 (Secondary): 연관 요소 (복수 가능)
- 핵심 성경 구절
- 설교 요약 (200자)
- 적용점 키워드

### Phase B: 교재 생성 시스템
```
입력: classified/{요소}/*.json
처리: 12주 커리큘럼 구조로 편집·가공
출력: curriculum/{요소}_12주_교재.md
```

## 4. 12주 교재 구조 (요소별 공통 프레임)

| 주차 | 주제 유형 | 내용 구성 |
|------|----------|----------|
| 1주차 | 개론 | 해당 요소의 성경적 의미와 4차원적 원리 소개 |
| 2~4주차 | 기초 | 핵심 개념 정립 (설교 발췌 + 말씀 묵상) |
| 5~8주차 | 심화 | 실제 적용 사례 (설교 스토리 + 토론) |
| 9~11주차 | 훈련 | 실천 과제 중심 (개인/그룹 적용) |
| 12주차 | 통합 | 전체 복습 + 삶의 변화 나눔 |

### 주차별 교재 구성 요소
각 주차는 다음 구조로 작성:
1. **이번 주 핵심 말씀** (성경 구절)
2. **도입 질문** (마음 열기, 2~3개)
3. **본문 강의** (조용기 목사 설교 발췌 + 해설, 1,500자)
4. **말씀 연구** (관련 성경 구절 5개 + 주석)
5. **토론 질문** (소그룹 나눔, 4~5개)
6. **실천 과제** (주간 적용, 개인/가정)
7. **기도문** (마무리 기도)

## 8. 개발 단계 및 우선순위

### 1단계 (MVP): 생각 요소 교재 완성
1. 설교 분류 스크립트 개발
2. '생각' 관련 설교 100편 분류 테스트
3. '생각' 12주 교재 1개 생성
4. 품질 검증 및 개선

### 2단계: 나머지 3개 요소 교재
- 믿음, 꿈, 말 순서로 반복 적용

### 3단계: 시스템 완성
- 전체 1,000편 자동 분류
- 4개 교재 완성 및 포맷 통일
