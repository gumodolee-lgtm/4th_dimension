# 4차원의 영성 설교 분류 리포트

## 개요

조용기 목사 설교 전체를 **4차원의 영성** 4대 요소(생각/꿈/믿음/말)로 자동 분류한 결과입니다.

- **분류기 버전**: v2.1-LLM (키워드 스코어링 + Claude LLM 검증)
- **분류 일시**: 2026-03-25
- **총 파싱 설교**: 1,526편 (1981~2020년)
- **최종 검증 정확도**: 91.3% (기존 수동 분류 286편 대비, 261편 일치)

## 2단계 분류 방법론

### 1단계: 키워드 가중치 스코어링

각 설교를 4개 카테고리에 대해 점수를 매기고, 최고 점수 카테고리를 주요 분류로 지정합니다.

**스코어링 규칙:**
| 요소 | 가중치 | 정규화 |
|------|--------|--------|
| 제목 키워드 매칭 | 30점/건 | 없음 (제목 = 최강 신호) |
| 본문 핵심 키워드 (high) | 5점/건 | 텍스트 길이 정규화 |
| 본문 중간 키워드 (mid) | 3점/건 | 텍스트 길이 정규화 |
| 복합 정규식 패턴 | 8점/건 | 텍스트 길이 정규화 |

**분류 임계값:**
- 주요 분류 최소 점수: 30점
- 보조 분류: 주요의 60% 이상 & 최소 20점

### 2단계: LLM 경계 사례 재분류

키워드 분류기의 경계 사례 258편을 Claude API(Haiku 4.5)로 재분류하여 오버라이드 적용.

**경계 사례 기준:**
- Ground truth 불일치: 44건
- 4차원 분류 중 저점 (30~45점): ~100건
- 기타 분류 중 고점 (23~30점): ~100건

**LLM 분류 조건:**
- 확신도 high 또는 medium인 경우만 오버라이드 적용
- 결과: **159편 변경** (258편 중)

### 텍스트 길이 정규화
`norm_factor = 10,000 / body_length` — 긴 설교가 짧은 설교보다 유리하지 않도록 보정.

### 인코딩 자동 감지
파일별로 utf-8, utf-8-sig, cp949, euc-kr 순서로 시도하여 인코딩이 다른 파일들을 정확히 읽음.

### LLM 재분류 주요 수정 사례

| 제목 | 변경 | LLM 판단 사유 |
|------|------|--------------|
| 죽은 자와 방불한 사람 | 기타→**믿음** | 아브라함의 신앙 여정, 믿음의 시련과 단계 |
| 믿는 방법 | 꿈→**믿음** | 하나님을 믿고 의심하지 않는 믿음의 조건 |
| 축복된 생활태도 | 꿈→**생각** | 마음의 생각 자세가 운명 좌우, 긍정적 사고 |
| 믿음에 굳게 서는 길 | 말→**믿음** | 아브라함의 신앙, 말씀에 대한 확신 |
| 성공으로 가는 7가지 이정표 | 기타→**생각** | 마음의 평화, 태도를 통한 성공 |
| 하나님, 나 그리고 세상 | 꿈→**기타** | 일반 기독교 설교, 4차원 요소 불명확 |

## 남은 불일치 (25건, 8.7%)

- 대부분 수동 분류와 LLM 분류의 주관적 판단 차이
- 한 설교가 여러 4차원 요소를 복합적으로 다루는 경우 (예: 믿음+꿈)
- 보조 설교("보조" 접두어) 중 원본과 다른 맥락에서 재해석된 경우
