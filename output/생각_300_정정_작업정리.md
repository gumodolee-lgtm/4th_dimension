# 생각 300 정정 작업 정리

## 1. 작업 목적

- 클로드코드로 분류된 `생각` 관련 설교 300편을 다시 검증했다.
- 분류 점수, 신뢰도, 본문 중심 주제, 상위/하위 경계 사례를 함께 점검했다.
- 검증 결과를 바탕으로 `정정된 300개 개별 파일 폴더`를 새로 생성했다.

## 2. 검증 방법

- 기준 데이터:
  - `sermon/classification_v3_result.json`
  - `output/생각_개별` 내 기존 300개 개별 파일
- 검증 방식:
  - `생각` 상위 300편의 점수와 신뢰도를 재확인
  - `all_scores` 최고 카테고리와 실제 분류 카테고리의 일치 여부 확인
  - 컷오프 근처 설교와 저신뢰 설교를 직접 읽고 수동 판정
  - 유지 / 강제 제외 / 재검토 후 제외 우선 / 대체 후보로 구분

## 3. 검증 결과 요약

- 원본 상위 300편: 300편
- 검증 통과: 292편
- 강제 제외 권고: 4편
- 재검토 후 제외 우선: 4편
- 총 교체 수: 8편
- 평균 신뢰도: 0.513
- 상위 300 안에서 전체점수 최고 카테고리 불일치: 4편
